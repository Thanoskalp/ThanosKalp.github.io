Memory Game

A simple memory game with cards playable in 3 difficulties

This was the first project I had to create from zero in Java. I worked with on of my best friends and colleague Kostas Alexandris. The UI is very simple because it was not the point of the project. The Whole point of the project was to use correctly all the object oriented principles(encapsulation, abstruction, inheritance and polymorphism).

About the game

Launching the game you had to insert your name and choose a difficulty. The 3 difficulties were Easy (4x3), Medium (5x3), and Hard(6x6). The game is played 2 turns, the player goes first and the computer second. Whenever you find a matching pair you get 10 points. Whoever gets the most points wins. There is also a 3 minute time. If the timer ends and the pairs were not found, the player with the highest score wins. All the data (Name and score) is stored in an xml
